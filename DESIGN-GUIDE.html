  • ✦ --blue → CTAs, links ativos, ícones primários, bordas de foco

  • ✦ --text-muted → subtítulos, parágrafos secundários, labels

14 — Animações
Keyframes e Efeitos

Animações sutis que dão vida sem distrair. Três tipos principais: float, pulse-glow e fade-up (scroll reveal).

⚡
float
3–4s ease-in-out ∞
pulse-glow
6–8s ease-in-out ∞
fade-up
opacity: 0 → 1
translateY(28px → 0)
duration: 0.65s
delays: .1s a .6s
/* Float — ícones decorativos no hero */ @keyframes float { 0%, 100% { transform: translateY(0px); } 50% { transform: translateY(-12px); } } /* Pulse glow — blobs de fundo */ @keyframes pulse-glow { 0%, 100% { opacity: 0.4; } 50% { opacity: 0.65; } } /* Fade up — scroll reveal */ .fade-up { opacity: 0; transform: translateY(28px); transition: opacity .65s ease, transform .65s ease; } .fade-up.visible { opacity: 1; transform: translateY(0); } .delay-1 { transition-delay: .1s; } /* até .delay-6 (.6s) */ /* Transition padrão */ --transition: .25s cubic-bezier(.4, 0, .2, 1);
15 — Layout & Grid
Sistema de Grid e Breakpoints

Grid de 2 e 3 colunas com gap de 24px. Colapsa progressivamente em mobile.

📱 Breakpoints

≤ 1024px .grid-3 → 2 colunas
≤ 768px .grid-2/.grid-3 → 1 coluna · section 64px · btn menor
≤ 900px Hero grid, dark-section, categories → 1 coluna
≤ 480px .grid-3 → 1 coluna
.container { max-width: 1200px; margin: 0 auto; padding: 0 24px; } .grid-2 { display: grid; grid-template-columns: repeat(2, 1fr); gap: 24px; } .grid-3 { display: grid; grid-template-columns: repeat(3, 1fr); gap: 24px; } .section { padding: 100px 0; } .section-sm { padding: 60px 0; } /* Hero grid customizado */ grid-template-columns: 1fr 1fr; gap: 64px; align-items: center;

17 — CSS Tokens
Todos os Custom Properties

Copie e cole no :root do seu projeto para replicar toda a identidade visual.

:root { /* ── Cores ── */ --bg: #fffdf5; --bg-card: #ffffff; --bg-muted: #efeee9; --dark: #3e3a53; --blue: #3367ca; --blue-mid: #5585d8; --blue-light: #94b0ed; --blue-pale: #dce8f9; --lime-green: #e3fc87; --amber: #e8a14e; --amber-pale: #fdf0df; --text: #2d2b3f; --text-muted: #8a88a0; --white: #ffffff; /* ── Border Radius ── */ --radius-xs: 8px; --radius-sm: 12px; --radius: 20px; --radius-lg: 28px; --radius-xl: 40px; /* ── Sombras ── */ --shadow-sm: 0 2px 12px rgba(62, 58, 83, 0.06); --shadow: 0 4px 24px rgba(62, 58, 83, 0.08); --shadow-md: 0 8px 40px rgba(62, 58, 83, 0.12); --shadow-lg: 0 16px 60px rgba(62, 58, 83, 0.16); /* ── Misc ── */ --transition: 0.25s cubic-bezier(0.4, 0, 0.2, 1); --font: 'Plus Jakarta Sans', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif; --max-w: 1200px; }
